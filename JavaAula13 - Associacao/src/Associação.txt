ASSOCIAÇÃO :

 - ASSOCIAÇÃO É A RELAÇÃO ENTRE DOIS OBJETOS,  PRINCIPIOS TAMBEM APLICADOS EM BANCO DE DADOS RELACIONAL.
 - A ASSOCIAÇÃO PODE TER DUAS FORMAS DE APLICAÇÃO UNIDIRECIONAL E BIDIRECIONAL.
 UNIDIRECIONAL: É QUANDO UM OBJETO SO TEM UMA UNICA DIREÇÃO PARA REFERENCIA UM PARA MUITOS / MUITOS PARA UM.
 UM PARA MUITOS → QUANDO UMA CLASSE ASSOCIA UM OBJETO DE REFERENCIA DE OUTRA CLASSE QUE POSSUI MAIS DE UM OBJETO.
 MUITOS PARA UM → QUANDO UMA OUTRA CLASSE REFENCIA MAIS DE UM OBJETOS DELA PARA UM UNICO OBJETO DE UMA CLASSE.
 
 BIDIRECIONAL: FUNCIONA COMO REFERENCIA PARA AMBAS AS CLASSES ONDE O FUNCIONAMENTO UNIDIRECIONAL PODE ATUAR TANTO DE UM 
 LADO PARA OUTRO SEGUINDO O PRINCIPIO E REGRAS DA REFERENCIA UNIDIRECIONAL.
