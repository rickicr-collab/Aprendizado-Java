CLASSES GERENICAS:


 - PARA SE DEFINIR EM TEMPO DE COMPILAÇÃO O TIPO DEFINIDO PARA SER USADO NUMA CLASSE EXTERNA É A NOTAÇÃO <T>.
 - A NOTAÇÃO <T> // TYPE → SIGNIFICA QUE VOCÊ IRÁ PASSAR UM TIPO PARA A CLASSE ESPECIFICA
   DIFERENTES DE QUANDO SE TRABALHA COM COLEÇÕES QUE É A NOTAÇÃO <E>.
   
 - COM ISSO COMO FOI DEFINIDO ATRAVES DA NOTAÇÃO <T> NÃO SE PODE CRAIR UM BLOCO DE INICIALIZAÇÃO COM VALORES 
 ESPECIFICOS DE UMA CLASSE POIS NÃO SE SABE QUE TIPO DE VALOR SERÁ TRABALHADO , COM ISSO PODE SER TRABALHADO 
 COM CONSTRUTORES DE CLASSES 
 
 - É POSSIVEL TAMBEM CRIAR CLASSES COM MAIS DE UM ATRIBUTO INSERINDO SEUS ATRIBUTOS E CONTRUTORES RESPECTIVOS.
