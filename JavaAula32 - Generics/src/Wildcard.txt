WILDCARD:

 - É UM CARACTER QUE É ESPECIFICADO QUANDO SE QUER TRABALHAR COM ADICOES DE OBJETOS ESPECIFICANDO SE UM OBJETO
 PERTENCE A UMA SUPER CLASSE OU SUBCLASSES DESSA MESMA SUPERCLASSE E ADICIONA-LAS NORMALMENTE DENTRO DA LISTA.
 - O CARACTER CORINGA PARA REALIZAR ESSA OPERAÇÃO É ("?") → SINAL DE INTERROGAÇÃO.
 
 OBS: DEVE -SE TER CUIDADO QUE AO SE ATRIBUIR NOS PARAMETROS O CARACTER CORINGA PARA CADA SUBCLASSE DE SUPER 
 CLASSE PRÉ CRIADA NÃO SE DEVE ADICIONAR NENHUM OBJETO SE CASO VOCÊ ATRIBUA ALGUM MÉTODO PARA ADICIONAR 
 ALGUM NOVO OBJETO É GERADO UM ERRO EM TEMPO DE COMPILAÇÃO INCOMUN DEVIDO A JVM NÃ ENTENDER A SUA AÇÃO DEVIDO
 PROPRIEDADES DA FUNÇÃO DO WILDCARD.
 
 OBS2: UTILIZANDO O CARACTER CORINGA MAIS A PALAVRA SUPER É POSSIVEL CRIAR MÉTODOS PARA ADICIONAR ITENS DENTRO 
 DE UMA LISTA SENDO UM OBJETO DE DE SUA PROPRIA CLASSE OU DE SUA SUPERCLASSE.
 
 OBS3: NÃO É POSSIVEL UTILIZAR DENTRO DE UMA LINHA DE PARAMETRO O CARACTER CORINGA MAIS A PALAVRA IMPLEMENTS
 INDEPENDENTE SE ESTA SE TRABALHANDO COM CLASSE (ABSTRATA , CONCRETA OU INTERFACE) A PALAVRA QUE DEVES SER 
 USADA COMO PARAMETRO É EXTENDS.
