INTERFACE QUEUE:

 - O MÉTODOS QUEUE CRIA UMA LISTA A PARTIR DOS VALORES ADICIONADOS A O OBEJETO INSTANCIADO.
 OBS: SE VOCÊ IMPLEMENTAR UM OBJETO DE UMA CLASSE QUE NÃO IMPLEMENT O MÉTODO COMPARABLE É GERADO 
 EM TEMPO DE EXECUÇÃO E EXCEÇÃO CLASSCASTEXCEPTION.
 
 METODOS QUEUE:
  - REMOVE(); MOSTRA O PRIMEIRO OBJETO DENTRO DE UMA QUEUE E DEPOIS O REMOVE NO ENTANTO SE A QUEUE ESTIVER VAZIA ELE RETORNA
  UMA EXCEÇÃO.
  - PEEK(); MOSTRA O PRIMIERO OBJETO DENTRO DE UMA QUEUE MAIS NÃO REMOVE COMO O MÉTODO REMOVE().
  - POLL(); MOSTRA O PRIMIERO OBJETO DENTRO DE UMA QUEUE MAIS UTILIZA O MESMO PRINCIPIO DO MÉTODO PEEK() MAIS 
  REMOVE O OBJETO DENTRO DA COLEÇÃO.
  - OFFER(STRING E); É UM METODO QUE QUE VOC^TENTAR ADICIONAIS UM OBJETO NUMA QUEUE COM CAPACIDADE PRÉ DEFINIDA.
