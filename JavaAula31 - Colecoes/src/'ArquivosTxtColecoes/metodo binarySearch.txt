MÉTODO BYNARIO PARA BUSCAS EM ARRAYS E LIST:

 - USANDO O MÉTODO CONTAINSALL() É POSSIVEL VERIFICAR DENTRO DE UM ARRAY OU LIST SE AQUELE OBJETO ESTÁ 
   CONTIDO DENTRO REALIZANDO UMA BUSCA.
 - USANDO O BINARYSEARCH() ELE RETORNA O INDICE DO VALOR OU OBJETO SE ENCONTRADO DENTRO DO ARRAY OU LIST
   SE NÃO FOI ENCONTRADO ELE MOSTRA O INDICE NO QUAL DEVERIA TER SIDO INSERIDO O VALOR OU OBJETO NAO ACHADO.
 - REGRAS A SER APLICADAS USANDO O MÉTODO BINARYSEARCH:
   - 1ª REGRA : PARA UTILIZAR O MÉTODO BINARYSEARCH A LISTA A SER USADA PARA A BUSCA JA DEVE ESTÁ ORDENADA.
   - 2ª REGRA : ORDEM PARA INSERÇÃO QUE REPRESENTA DESSA FORMA (-(PONTO DE INSERÇÃO) -1);
      * POR EXEMPLO SE O RETORNO DA BUSCA FOR ZERO NÃO HÁ COMO VERIFICAR SE O RETORNO É O VALOR DO INDICE.
   - 3ª REGRA : SE FOI UTILIZADO UM COMPARATOR O MÉTODO BINARYSEARCH POSSUI UM MÉTODO SOBRECARREGADO QUE ELE
   RECEBE COMPARATOR.
   - 4ª REGRA : TANTO USANDO COLLECTIONS COMO USANDO ARRAYS O RESULTADO DA BUSCA É SEMPRE O MESMO.
