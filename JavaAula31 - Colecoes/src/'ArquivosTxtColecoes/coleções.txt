INTRODUÇÃO A COLEÇÕES

- AS COLEÇÕES NO JAVA POSSUEM ALTA COESÃO: QUANDO UMA CLASSE É BEM ESPECIFICA E REALIZAR SEUS PROPOSITOS
	SEM MISTURAR OS PROPROSITOS DE OUTRAS CLASSE.
- COLEÇÕES É UMA INTERFACE QUE EXTENDE SUAS FUNCIONALIDADES PARA OUTRAS CLASSES COMO POR EXEMPLO:
	(SET, LIST, QUEUE) TODAS POSSUEM MÉTODOS PARA SE TRABALHAR AS COLEÇÕES.
- EXISTEM DOIS TIPOS DE COLEÇOES:
	* ORDENADAS → ORDENED
	* CLASSIFICADAS OU SORTIDAS → SORTED
