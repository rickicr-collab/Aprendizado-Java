REMOVENDO ITENS DE UMA LISTA USANDO MÉTODO ITERATOR:

MÉTODO ITERATOR É UMA FORMA EFICIENTE DE PECORRER LISTAS E EXECUTAS FUNÇÕES BASICAS SUPORTADAS POR ELAS.

- VOCÊ IRÁ CRIAR UMA VARIAVEL DE REFERENCIA PARA A INTERFACE ITERATOR.
- E IRÁ UTILIZAR A VARIAVEL COM O MÉTODO ITERATOR ATRAVESZ DE UM WHILE UTILIZANDO ALGUNS MÉTODOS DENTRO DO SEU BLOCO.
  - HASNEXT(): MÉTODO QUE VERIFICA SE EXISTE UM PROXIMO ELEMENTO NA LISTA.
  - NEXT(): MÉTODO QUE VAI PEGAR O ELEMENTO DA LISTA TRABALHADA.
