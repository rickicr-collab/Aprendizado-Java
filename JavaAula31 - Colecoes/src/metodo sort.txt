METODO SORT 

METODO SORT É UTILIZADO PARA ORDENAR LISTA E VALORES EM QUANTIDADES MUITO GRANDES OU PARA FILTRO
DE DADOS

- DEVIDO A GRANDE QUANTIDADES DE VALORES DIFERENTES A SEREM FILTRADOS E ORDENADOS O JAVA CRIOU 
  UMA CLASSE CHAMADA COLLECTION QUE PODE RESOLVER TODOS OS METODOS E ATRIBUTOS PARA CADA TIPO DE 
  VALOR OU DADO A SER ORDENADO MUITO UTIL NO JAVA PARA ORDENAR ARRAYSLIST
- SE O ARRAYLIST JÁ ESTIVER ESPECIFICADO O TIPO DE VALOR INSTANCIADO AO CRIAR A LISTA O JAVA JÁ 
  AUTIMATIVA A ORDENAÇÃO DE FORMA SIMPLES E RAPIDA COMO POR EXEMPLO(STRING: ORDENDADO POR LETRAS
  ALFABÉTICAS, INTEGER / DOUBLE : POR VALORES NUMERICOS MENOR PARA O MAIOR).
- A CLASSE COLLECTION QUANDO INSTANCIA UM OBJETO DE UMA DETERMINADA CLASSE NÃO CONSEGUE INDENTIFICAR 
  QUAL FILTRO USAR DEVIDO A ISSO A CLASSE A QUAL FOI INTANCIADO UM OBJETO E ADICIONADO A LISTA DEVE
  SER IMPLEMENTADA COM A INTERFACE CHAMADA COMPARABLE E O MÉTODO COMPARE TO DEVE SER SOBREESCRITO DENTRO
  DA CLASSE COM ALGUMAS REGRAS PARA USO DO MÉTODO.
- REGRAS DO MÉTODO COMPARE TO:
  - // NEGATIVO : SE THIS.OBJETO < OUTRO OBJETO;
  - // ZERO: SE THIS.OBJETO == OUTRO OBJETO;
  - // POSITIVO: SE THIS OBJETO > OUTRO OBJETO;
- O MÉTODO COMPARE TO SO É NORMALMENTE UTILIZADO EM CLASSES WRAPPED , TIPOS PRIMITIVOS NÃO POSSUI OS MÉTODOS 
  EXISTEM DUAS OPÇÕES PARA RESOLVER ESSE PROBLEMA.
  1ª SOLUÇÃO: TRANSFORMA O ATRIBUTO PARA COMPARAÇÃO DE TIPO PRIMITIVO PARA UM TIPO WRAPPER
  USANDO A PRIMEIRA SOLUÇÃO APLICANDO DIRETAMENTE SOBRE O ATRIBUTO SEM PRECISAR CHAMAR ALGUM
  MÉTODO DEVIDO AO ENCAPSULAMENTO DO ATRIBUTO PORQUE O MAIS IMPORTANTE NO MÉTODO COMPARE TO É O VALOR.
  2ª SOLUÇÃO: É ATRIBUIR UMA VARIAVEL DE REFERENCIA DO TIPO DOBLE WRAPPER PARA UM ATRIBUTO DO TIPO
  PRIMITIVO DENTOR DO MÉTODO COMPARE TO 
- NO ENTANTO ESSE MÉTODO É CONSIDERADO ERRADO POIS DEVIDO AO CASO DE CLASSES JÁ ESTIVEREM PRÉ DEFINIDAS
  E ESTAREM SENDO USADAS POR OUTROS (EX: PROGRAMADORES) SE VOCÊ POR ACASO ALTERAR A FORMA DE COMPARAR DE
  DOUBLE PARA STRING IRÁ QUEBRAR O CODIGO DOS OUTROS SOBREESCREVENDO O MÉTODO DEVE SE TORMAR CUIDADO DEVIDO 
  A ISSO.
