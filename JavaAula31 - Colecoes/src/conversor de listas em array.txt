COMO CONVERTER LISTAS EM ARRAYS E VICE VERSA:

CONVERTENDO UMA LISTA EM ARRAY:
 - VOCÊ PODE UTILIZAR O MÉTODO TOARRAY() PARA CONVERTER UMA LISTA EM ARRAY.
 - OU CRIAR UM ARRAY DE OBJETOS APLICANDO O MÉTODO TOARRAY().
 - APOS REALIZADO ESSA TAREFA É SÓ IMPRIMIR EM TELA OU UTILIZAR O ARRAY PARA O PROPROSITO ESPECIFICO.

CONVERTENDO UM ARRAY EM LISTA:
 - VOCE VAI CRIAR UMA NOVA LISTA DE ACORDO COM SUA CLASSE E TIPO DE VALOR EXISTENTE NO ARRAY
 - E VAI ASSOCIAR ESSA LISTA A FUNÇÃO ARRAY.ASLIST().
 OBS:  COM O MÉTODO ASLIST();
  - CRIA UMA LISTA MAIS ACABA FICANDO NO MESMO LUGAR NO RIP ( AS ALTERAÇOES FEITAS EM UM ALTERAM NO OUTRO).
