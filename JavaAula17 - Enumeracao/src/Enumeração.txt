 - PARA EVITAR CASOS ONDE VOCE NÃO NECESSITA DE VARIAVEIS STATICAS FINAIS

 - AO CRIAR UM NOVO OBJETO NO METODO PRINCIPAL MAIN VOCE INSTANCIA O ATRIBUTO

 OBS3: VOCE TAMBEM PODE CRIAR UMA NOVA ENUMERAÇÃO DENTRO DE UMA CLASSE DESDE QUE SIGA AS REGRAS DE CONVERSÃO NORMAIS ESTABELECIDAS.
 OBS4: VOCE TAMBEM PODE COLOCAR MODIFICADORES DE ACESSO NAS ENUMERAÇÕES DECLARADAS EM UMA CLASSE.
 OBS5: NÃO É POSSIVEL OU PERMITIDO CRIAR UMA ENUMERAÇÃO DENTRO DE UM METODO O JAVA E A JVM IMPEDE A CRIAÇÃO DO MESMO.
 OBS6: COMO A ENUMERAÇÃO É UM TIPO DE CLASSE ESPECIAL É POSSIVEL ADICIONAR ATRIBUTOS E MÉTODOS A MESMA.
