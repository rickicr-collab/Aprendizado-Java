**OPERADORES**

 - OPERADORES ARITMETICOS SÃO OPERADORES QUE VOCÊ PODE REALIZAR OPERAÇÕES ARITMETICAS.
	* OPERADOR DE SOMA: (+)
	* OPERADOR DE SUBTRAÇÃO: (-)
	* OPERADOR DE MULTIPLICAÇÃO: (*)
	* OPERADOR DE DIVISÃO: (/)
	* OPERADOR DE RESTO: (%)
	OBS:  NAS OPERAÇÕES DE DIVISÃO VOCÊ DEVE ESTÁ ATENTO SE OS DOIS NUMEROS SÃO INTEIROS POIS SE O VALOR FOR RESULTADO 
	REAL O RESULTADO DE NUMEROS INTEIROS SÃO SERÁ EXIBIDO DE FORMA CORRETA POR ISSO SE DEVE FICAR ATENTO AOS TIPOS PRIMITIVOS 
	UTILIZADOS NAS OPERAÇÕES.
	OBS2: É POSSIVEL REALIZAR CASTING EM VARIAVEIS PARA QUE O VALOR SEJA CONVERTIDO ANTES DE REALIZAR A OPERAÇÃO E EXIBIR O 
	RESULTADO EM TELA MAIS NÃO FICA VISUALMENTE AGRADAVEL SEU CODIGO EMBORA SEJA POSSIVEL.
	
-----------------------------------------------
	
 - OPERADORES DE COMPARAÇÃO SÃO OPERADORES QUE SÃO UTILIZADOS PARA SE REALAIZAR OPERAÇÕES COMPARAVEIS:
 	* MENOR QUE: (<)
 	* MAIOR QUE: (>)
 	* MENOR OU IGUAL: (<=)
 	* MAIOR OU IGUAL: (>=)
 	* IGUAL A : (==)
 	* DIFERENTE DE: (!=)
 	
 	OBS: EXISTE UM PADRÃO NO JAVA PARA UTILIZAÇÃO DE OPERADORES DE COMPARAÇÃO SEMPRE QUE VOCÊ QUER UTILIZA-LOS 
 	DEVE-SE UTILIZAR A PALAVRA RESERVADA (is).
	OBS2: NÃO É POSSIVEL REALIZAR A OPERAÇÃO DE COMPARAÇÃO COM TIPO PRIMITIVOS DIFERENTES SERÁ GERADO UM ERRO DE COMPILAÇÃO.
	
-----------------------------------------------

  - OPERADORES LOGICOS:  SÃO OPERADORES PARA REALIZAR OPERAÇÕES LÓGICAS GERALMENTE DENTRO DE ESTRUTURAS DE CONDDIÇÃO.
   * AND: NO JAVA É REPRESENTADO POR (&&) - para que seja verdadeiro no and ambas as condições devem ser verdadeiras 
   * OR : NO JAVA É REPRESENTADO POR (||) - para que seja verdadeiro no or uma das condições deve ser verdadeiras
   * NOT: NO JAVA É REPRESENTADO POR (!) - 
   
-----------------------------------------------

  - OPPERADORES DE ATRIBUIÇÃO: SÃO OPERADORES QUE IRÃO ATRIBUIR ALGO A SUA VARIAVEL.
  	* OPERADOR DE ATRIBUIÇÃO RECEBE: (=)
  	* OPERADOR DE ATRIBUIÇÃO SOMA IGUAL: (+=)
  	* OPERADOR DE ATRIBUIÇÃO SUBTRAÇÃO IGUAL:(-=)
  	* OPERADOR DE ATRIBUIÇÃO MULTIPLICAÇÃO OU IGUAL: (*=)
  	* OPERADOR DE ATRIBUIÇÃO DIVISÃO OU IGUAL:(/=)
  	* OPERADOR DE ATRIBUIÇÃO POR PORCETAGEM OU IGUAL: (%=)
  	
  - OPERADORES DE ATRIBUIÇÃO PARA INCREMENTAÇÃO: SÃO OPERADORES QUE AJUDAM NA INCREMENTAÇÃO DE SUA VARIAVEL.
  	* OPERADOR DE ATRIBUIÇÃO PARA INCREMENTAÇÃO POSITIVA: (++)
  	* OPERADOR DE ATRIBUIÇÃO PARA INCREMENTAÇÃO NEGATIVA: (--)
  	
  	OBS: A ORDEM QUE É ATRIBUIDA OS OPERADORES AFETA O RESULTADO SE O OPERADOR NEGATIVO / POSITIVO FOI ATRIBUIDO ANTES DA VARIAVEL 
  	A OPERAÇÃO DE INCREMENTO É FEITA ANTES DO VALOR DA VARIAVEL / SE O OPERADOR FOI ATRIBUIDO APOS A VARIAVEL SERÁ MANTIDA
  	E DEPOIS SERÁ REALIZADO O INCREMENTO NEGATIVO OU POSITIVO.
